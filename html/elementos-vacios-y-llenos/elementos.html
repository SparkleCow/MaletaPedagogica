    Elementos llenos y vacíos

    Los elementos llenos son aquellos que pueden tener contenido, como texto, otros elementos HTML, imágenes u otros elementos multimedia dentro de sus etiquetas de apertura y cierre. Ejemplos de elementos llenos comunes incluyen <div>, <p>, <h1>, <a>, <ul>, <ol>, <li>, <table>, y muchos otros.

    Elementos vacíos

    Los elementos vacíos, a veces llamados "elementos autónomos", son aquellos que no pueden tener contenido entre sus etiquetas de apertura y cierre. Estos elementos no tienen una etiqueta de cierre y se suelen escribir como etiquetas auto-cerradas (self-closing tags) en HTML. Por ejemplo, <img>, <br>, <input>, <meta>. Los elementos vacíos se utilizan para incrustar contenido externo, como imágenes o para crear saltos de línea en el texto.
